Bienvenue dans les notes de ce nouveau patch !

Comme les récents patchs, le 7.17 est plutôt léger en équilibrages, mais n'allez pas croire qu'il est tout petit : il se passe plein d'autres choses ! Ornn émerge de ses siècles de solitude (pour devenir un top-laner, parce que la compagnie, ce n'est vraiment pas son truc) et cinq nouveaux Gardiens des étoiles atterrissent dans la Faille le Parc de Valoran. Vous vous demandez où se trouve le Parc de Valoran ? Alors attendez l'arrivée d'Invasion, le tout nouveau mode de jeu de League of Legends. Ce patch apporte également une amélioration bienvenue aux sets d'objets : vous pouvez enfin présélectionner les emplacements dans lesquels vous voulez placer les objets que vous achetez !

Dernière petite remarque : le patch 7.17 couvrira une période plus longue que d'habitude. Le prochain sera celui que nous prévoyons d'utiliser pour le Mondial.

Bonne chance à tous sur le champ de bataille !
